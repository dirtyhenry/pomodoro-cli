# ``Pomodoro``

## Overview

Features include:

- **Hooks**: Pomodoro can optionnaly run shell scripts when a pomodoro starts and/or finishes (samples are provided);
- **Journal**: Keeps your log entrys in a YAML file so you can know how you spend your time.
